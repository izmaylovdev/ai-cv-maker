## Product Summary

AI CV Maker is a web application that helps users create, manage, and export professional CVs.
Users can maintain multiple job profiles tailored to different target roles, leverage AI to improve their content,
and generate polished PDFs ready to send to employers.

### Core User Journey

```
Register / Log in
       ↓
Create a Job Profile  ←  (or upload an existing CV to bootstrap)
       ↓
Fill in / edit profile data
  • Personal info, work experience, education, skills
  • Use AI enhancement on individual fields at any time
       ↓
Optionally: Optimize entire profile for a specific target role (AI)
       ↓
Generate a tailored CV  (optionally add per-generation notes)
       ↓
Preview & download PDF
```
